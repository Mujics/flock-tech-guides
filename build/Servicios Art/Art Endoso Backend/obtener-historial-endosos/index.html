obtener-historial-endosos

Nombre del Servicio​

Consulta de endosos cualitativos

Endpoint​

GET /obtener-historial-endosos/{nroContrato}

Tipo​

GET

Pantalla​

ARTF51016

Consideraciones de Seguridad​

Se ingresa desde el botón Producción/Endosos en la pantalla de Consulta de pólizas. Hereda los permisos desde la pantalla ARTF51000.

Momento de Ejecución​

Al momento de ingresar a la pantalla. No se permite realizar ninguna operación de actualización, inserción u eliminación.

Descripción del Servicio​

Se consulta información de la tabla GIOSEG.CART_CERTIFICADOS_ENDOSOS para el número de contrato recibido como parámetro. Cuando se obtiene la información se consulta la descripción del motivo desde la tabla CART_MOTIVOS_ENDOSOS. Si el valor de CACW_CAME_CD_MOTIVO es 414, 418 o 419 entonces el valor del elemento TEXTO es devuelto por la función EMPRESA_SUBROGACION_ART con parámetros definidos por sucursal, ramo, número de póliza y número de endoso. Caso contrario el valor del elemento TEXTO es el valor del campo CACW_LINEA_ANEXOS. Luego se obtiene el identificador único del registro (ROWID) correspondiente a la sucursal, ramo, número de póliza, certificado = 0 y número de endoso. Para luego obtener el texto de la póliza con la función BUSCAR_TEXTO_POLIZA.

Observaciones adicionales​

Ok

Código encontrado o sugerido​

SELECT CACW_CAPO_NU_POLIZA, CACW_NU_ENDOSO, CACW_FE_DESDE, CACW_FE_OPERACION, CACW_CAME_TP_TRANSAC, CACW_CAME_CD_MOTIVO, CACW_CAAN_CD_ANALISTA, CACW_LINEA_ANEXOS
FROM GIOSEG.CART_CERTIFICADOS_ENDOSOS
WHERE CACW_CAPO_NU_POLIZA = NroContrato
AND CACW_CASU_CD_SUCURSAL = 2
AND CACW_CARP_CD_RAMO = 80
AND CACW_CAME_TP_TRANSAC IN ('L','A');

En Post-Query:
SELECT came_de_motivo
INTO :DESC_MOTIVO
FROM gioseg.cart_motivos_endosos
WHERE came_tp_transac = :CACW_CAME_TP_TRANSAC
AND came_cd_motivo = :CACW_CAME_CD_MOTIVO;

declare
 l_rowid rowid; 
 TEXTO VARCHAR2(144); 
begin
  SELECT rowid
  into l_rowid
  FROM gioseg.cart_polizas_textos
  where cpot_casu_cd_sucursal = :cacw_casu_cd_sucursal
  and cpot_carp_cd_ramo = :cacw_carp_cd_ramo
  and cpot_capo_nu_poliza = :cacw_capo_nu_poliza
  and cpot_cace_nu_certificado = 0
  and cpot_nu_endoso = :cacw_nu_endoso;

  :texto_endoso := BUSCAR_TEXTO_POLIZA(l_rowid);
  IF :CACW_CAME_CD_MOTIVO IN (414,418,419) THEN
    :TEXTO := EMPRESA_SUBROGACION_ART(:cacw_casu_cd_sucursal, :cacw_carp_cd_ramo, :cacw_capo_nu_poliza, :cacw_nu_endoso);
  ELSE
    :TEXTO := :CACW_LINEA_ANEXOS;
  END IF;
exception
  when others then
    :texto_endoso := null;
end;

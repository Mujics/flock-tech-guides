validar-productor-nuevo

Validar productor nuevo​

Endpoint​

/productor/findByCodigo/{codigo}

Tipo de servicio GET​

Consideraciones de seguridad​

Momento de ejecución​

Al momento de ingresar código de productor nuevo

Descripción del servicio​

"Si se ha ingresado un código de productor nuevo se consulta de tabla GIOSEG.CART_PRODUCTORES el nombre del productor y el código de esquema del nuevo productor.
Si el código de esquema es NULO se debe mostrar el mensaje 'El esquema del productor nuevo debe existir' y se cancela el proceso."

Observaciones varias​

"Verificar servicio de validación anterior: En caso que no tenga validación de esquema solicitar el agregado.
En este caso, como no existe el dato ESQUEMA, hay que modificar el endpoint:
/productor/findByCodigo/{codigo}, agregándolo al final del JSON (consultar)."

Código encontrado o sugerido​

if :PRODUCTOR_NUEVO is not null then  
  DSP_PRODUCTOR_NUE( :PRODUCTOR_NUEVO, :NOMBRE_PRODUCTOR, :ESQUEMA);  
  IF :ESQUEMA IS NULL THEN  
    MESSAGE ('El esquema del productor nuevo debe existir');  
    raise form_trigger_failure;  
  END IF;  
end if;  
PROCEDURE DSP_PRODUCTOR_NUE(v_codigo IN cart_productores.capd_cd_productor%type,  
                        v_dsp IN OUT cart_productores.capd_nm_productor%type,  
                        v_esq IN OUT cart_productores.capd_cres_cd_esquema%type) IS  
BEGIN  
  IF v_codigo IS NOT NULL THEN  
    begin  
      SELECT CAPD_NM_PRODUCTOR,  
             CAPD_CRES_CD_ESQUEMA  
      INTO v_dsp, v_esq  
      FROM CART_PRODUCTORES  
      WHERE CAPD_CD_PRODUCTOR = v_codigo;  
    exception  
      when no_data_found then  
        v_dsp := '???';  
        v_esq := '???';  
    end;  
  END IF;  
END;
